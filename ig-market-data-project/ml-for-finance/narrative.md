# ml-for-finance — Machine Learning for Finance

## Objective
Apply ML to financial time series: regime detection, return forecasting,
sentiment-aware models, and deep learning for alpha generation — all trained
and evaluated on IG market data.

## Projects (Planned)
1. **Regime Detection** — HMM / GMM for bull/bear/range classification
2. **Return Forecasting** — XGBoost / LightGBM on technical + macro features
3. **LSTM Price Predictor** — Sequence-to-sequence for short-term price
4. **Sentiment-Augmented Model** — NLP sentiment + price features
5. **Model Interpretability** — SHAP / LIME for each ML prediction
